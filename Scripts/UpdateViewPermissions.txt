## Quick Template for Adding Permission Checks to Views

### For Create/Add buttons:
Replace:
<button>Add New</button>

With:
@if (HasPermission("ControllerName", "Index", "Create"))
{
    <button>Add New</button>
}

### For Edit buttons:
Replace:
<button onclick="edit(id)">Edit</button>

With:
@if (HasPermission("ControllerName", "Index", "Edit"))
{
    <button onclick="edit(id)">Edit</button>
}

### For Delete buttons:
Replace:
<button onclick="delete(id)">Delete</button>

With:
@if (HasPermission("ControllerName", "Index", "Delete"))
{
    <button onclick="delete(id)">Delete</button>
}

### For Export buttons:
Replace:
<button onclick="export()">Export</button>

With:
@if (HasPermission("ControllerName", "Index", "Export"))
{
    <button onclick="export()">Export</button>
}
